Digital Twin Dataset API

Access the API by making requests to {{api_url}}.

For example, after cloning github.com/caltech-netlab/digital-twin-dataset, run the following Python code: 

from dataset_api_client import DatasetApiClient

data_api_client = DatasetApiClient(base_url="{{api_url}}")
data_api_client.download_data(
    magnitudes_for=[<element names>],
    phasors_for=[<element names>],
    waveforms_for=[<element names>],
    time_range=(<start>, <end>),
    resolution=<duration>,
)

See github.com/caltech-netlab/digital-twin-dataset for more details.
